Comments deletion is a plugin that quickly removes all comments and resets the comment count icon. Administrators can quickly remove ALL comments...simple.




== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on the 'Setting' option and then click on 'Comments Deletion'.
